    7. 7. Testing Scenario First tests conducted by AcceDE in France. Complementary testing done by AccessibilitéWeb in Québec. Developed specific PDF test files (tagged, partially tagged, untagged), to measure different test categories. No prior determination of the accessibility level of tested PDF files. No specific SEO strategy or optimization changes brought to the PDF files.2012. AccessibilitéWeb Toll Free: 1 (877) 315-5550 7
    8. 8. Testing Tools PDF converted from OpenOffice Writer 3.3.0 and final tagging with Acrobat Pro X. Indexing with Google Web Master Tools. Search with Google on specific domain name, with specific file extension (.pdf).2012. AccessibilitéWeb Toll Free: 1 (877) 315-5550 8
    9. 9. Testing Categories • Titles • Main content language • Replacement text • Semantic structure tagging • Reading content order • Metadata • Language changes • Color contrast • Complex Table2012. AccessibilitéWeb Toll Free: 1 (877) 315-5550 9
    10. 10. Testing Data Different files sets for each category, including: • Basic text : litteral English background text. • Target text: short technical English text meant to be found by Google, without significant common words within the basic text. • French quote: short French paragraph of text to be found by Google.2012. AccessibilitéWeb Toll Free: 1 (877) 315-5550 10
    11. 11. Reporting Structure OK : Indexed file and target text found. NO : Indexed file and target text not found. N/A : File not indexed.2012. AccessibilitéWeb Toll Free: 1 (877) 315-5550 11

    13. 13. Titles Files set 1. No tag, no title. 2. No tag, title. 3. Tagged file, no title 4. Tagged file, title2012. AccessibilitéWeb Toll Free: 1 (877) 315-5550 13
    14. 14. Titles Results Title is used by Google and displayed in the results page.2012. AccessibilitéWeb Toll Free: 1 (877) 315-5550 14
    15. 15. Main content language Is Google taking into consideration the specified main content language?2012. AccessibilitéWeb Toll Free: 1 (877) 315-5550 15

    17. 17. Main content language Results Google does not take into consideration the specified language.2012. AccessibilitéWeb Toll Free: 1 (877) 315-5550 17
    18. 18. Replacement text Is Google reading the replacement text?2012. AccessibilitéWeb Toll Free: 1 (877) 315-5550 18

    20. 20. Replacement text Results Google does not read the replacement text.2012. AccessibilitéWeb Toll Free: 1 (877) 315-5550 20
    21. 21. Semantic structure tagging Is Google taking into consideration the semantic structure in tags?2012. AccessibilitéWeb Toll Free: 1 (877) 315-5550 21

Tagged file, reading order = “ordrematusbalatum belli”, reflow order = “balatum belli ordrematus”.2012. AccessibilitéWeb Toll Free: 1 (877) 315-5550 25

Basic text with correct tags, no Author, no Summary. Test scenario Searching Target text. Searching modification date.2012. AccessibilitéWeb Toll Free: 1 (877) 315-5550 28
    29. 29. Metadata Results The other metadata have no impact on the search.2012. AccessibilitéWeb Toll Free: 1 (877) 315-5550 29
    30. 30. Language changes Does indicating the language changes help to find content?2012. AccessibilitéWeb Toll Free: 1 (877) 315-5550 30

Basic text including the French quote, basic and French tagged. Test scenario Searching French quote.2012. AccessibilitéWeb Toll Free: 1 (877) 315-5550 31
    32. 32. Language changes Results In our test, indicating language change helps to find the entire French quote.2012. AccessibilitéWeb Toll Free: 1 (877) 315-5550 32

AccessibilitéWeb Toll Free: 1 (877) 315-5550 34

    36. 36. Colour contrast Results Color contrast have to impact.2012. AccessibilitéWeb Toll Free: 1 (877) 315-5550 36

Target text in plain text. 4. Target text in a complex table, with tag and explicit link between data cells and headers.2012. AccessibilitéWeb Toll Free: 1 (877) 315-5550 38
    39. 39. Complex Table Test scenario Searching Target text.2012. AccessibilitéWeb Toll Free: 1 (877) 315-5550 39
    40. 40. Complex Table Results No impact at all.2012. AccessibilitéWeb Toll Free: 1 (877) 315-5550 40
    41. 41. Summary 1. Are the accessibility guidelines helping the PDF files being indexed by search engines? Very little help. Yes for titles, reading content order. No for main content language, replacement text, semantic structure tagging, metadata, language changes, color contrast, complex table.2012. AccessibilitéWeb Toll Free: 1 (877) 315-5550 41
    42. 42. Summary 2. Can accessibility guidelines hurt indexing? No.2012. AccessibilitéWeb Toll Free: 1 (877) 315-5550 42
